iburg, A Tree Parser Generator

iburg is a program that generates fast tree parsers for cost-augmented tree grammars. iburg is useful for writing code generators and for teaching computer science compiler courses. A variant of iburgis used in the code generators for lcc, a retargetable compiler for ANSI C. iburg is described in

Chris Fraser, D. R. Hanson, and T. A. Proebsting, `Engineering a Simple, Efficient Code Generator Generator,' ACM Letters on Programming Languages and Systems 1, 3 (Sep. 1992), 213-226.

Compressed Postscript (59748 bytes) for this paper is available. The source code is available in iburg.tar.gz (146228 bytes) or iburg.zip (154504 bytes); the README file elaborates. The LOG file describes recent changes. You can also read the man page (an ASCII file) and browse the source code.

There is an iburg mailing list. To be added to the list, send a message with the 1-line body

subscribe iburg

to majordomo@cs.princeton.edu. This line must appear in the message body; `Subject:' lines are ignored. To learn more about mailing lists served by majordomo, send a message with the body `help' to majordomo@cs.princeton.edu.

Mail sent to iburg@cs.princeton.edu is forwarded to everyone on the mailing list.
